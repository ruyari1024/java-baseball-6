# 📚기능 요구 사항

## 💻컴퓨터
- [ ] 게임 시작
- [ ] 3자리의 숫자의 중복이 없는 수 생성
- [ ] 사용자 입력 값과 정답 비교 후 결과 및 힌트 제공
  - 같은 값이 같은 위치에 있을 시 '스트라이크'
  - 같은 값이 다른 위치에 있을 시 '볼'
  - 같은 값이 전혀 없을 시 '낫싱'
- [ ] 게임 종료
## 🙍‍♀️사용자
- [ ] 3자리의 숫자의 중복이 없는 수 입력
- [ ] 사용자의 수가 유효한지 파악
  - 문자 입력
  - 길이 초과
  - 중복되는 숫자
- [ ] 유효한 수가 아닐 때 에러 발생
  - IllegalArgumentException 발생
- [ ] 정답 기입 시 게임 지속 혹은 게임 종료 선택
